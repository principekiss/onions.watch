### Install the Tor Browser

Debian or Ubuntu

```sh
sudo apt install tor -y
```

RHEL, CentOS or Fedora

```sh
sudo dnf install tor -y
```

Archlinux and other arch-based distribution

```sh
sudo pacman -S tor
```

### Create a Virtual Environement


```sh
pipenv shell
```
Install Dependencies

```sh
pipenv install
```

### Run the app
  
```sh
python app.py
```
